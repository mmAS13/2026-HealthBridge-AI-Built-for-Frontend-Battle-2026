# HealthBridge AI 🩺

AI-powered healthcare landing page concept — built for Pakistan and beyond. Check symptoms, find nearby doctors, and track your health, all from one screen.

**Live demo:** [aihealthbridge.netlify.app](https://aihealthbridge.netlify.app/)

## Features

- 🤖 **AI Symptom Checker** — pick symptoms or describe them in your own words, get instant guidance
- 👨‍⚕️ **Find Nearby Doctors** — browse verified doctors with ratings and one-click booking
- 📊 **Health Dashboard** — track vitals (heart rate, blood pressure, blood sugar, SpO2) and weekly activity
- 💊 **Medicine Reminders** — never miss a dose
- 🌐 **Urdu + English Support**
- 📱 Fully responsive, single-file build

## Tech Stack

- HTML5, CSS3 (custom properties / design tokens), vanilla JavaScript
- No build step, no dependencies — just open `index.html`
